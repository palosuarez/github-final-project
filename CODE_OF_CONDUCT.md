# Code of Conduct

## Our Pledge
We want this project to be a respectful place where people can build together.
Everybody is welcome here, no matter background, identity, or experience level.

## Our Standards
What helps this community:

- Be respectful in issues, PRs, and reviews.
- Discuss ideas, not people.
- Be open to feedback and willing to improve.
- Help others when they’re stuck.

What is not okay:

- Harassment, hate speech, or discrimination.
- Insults, personal attacks, or hostile behavior.
- Sharing someone’s private info without permission.
- Repeated disruptive behavior.

## Enforcement Responsibilities
Maintainers are responsible for enforcing this Code of Conduct.
They can edit, remove, or reject comments, commits, issues, and pull requests that break these rules.

## Scope
This applies in all project spaces: issues, pull requests, discussions, and code reviews.
It also applies when someone is representing this project in public spaces.

## Enforcement
If you see or experience unacceptable behavior, report it to the maintainers.
We’ll review every report and handle it as fairly and quickly as possible.

## Enforcement Guidelines
### 1. Correction
For minor issues, maintainers will give a private warning and ask for a correction.

### 2. Warning
For repeated or more serious issues, maintainers will issue a formal warning.

### 3. Temporary Ban
For serious or repeated violations, maintainers may apply a temporary ban.

### 4. Permanent Ban
For severe harassment, threats, or sustained harmful behavior, maintainers may apply a permanent ban.
