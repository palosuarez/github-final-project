# Contributing

All contributions, bug reports, bug fixes, documentation improvements, enhancements, and ideas are welcome.

This project started as part of a Git and GitHub learning path, but the goal is practical: keep the repo clean, readable, and useful for anyone reviewing or extending it.

If you find a bug, open an issue with clear steps to reproduce it.
If you already have a fix, open a pull request with a short explanation of what changed and why.

Small improvements are just as valuable as big ones.
